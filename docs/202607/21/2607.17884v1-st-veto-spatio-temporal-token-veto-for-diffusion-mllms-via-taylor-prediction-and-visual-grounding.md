## 摘要
视觉语言模型（VLM）通过思维链（CoT）提示实现了强推理，但存在高顺序生成成本、错误累积和有限的自我纠正。扩散多模态大语言模型（dMLLM）以顺序无关的方式解开令牌，提高了效率并实现了迭代细化，然而其推理能力及如何增强仍未被充分探索。我们提出了一种无需训练的方法——空间-时间令牌否决（ST-Veto），该方法利用每个扩散步骤中观察所有令牌位置的能力。ST-Veto不仅依赖于当前步骤的置信度，还通过二阶泰勒预测置信度动态来否决时间上不稳定的令牌，并使用图像注意力质量过滤弱基础令牌，将其替换为更安全的候选令牌。在多个dMLLM和多模态推理基准上，ST-Veto始终优于标准解码策略和先前的VLM推理方法，在无需额外训练或生成成本的情况下，准确率提升高达9%。分析表明，ST-Veto将生成导向更高置信度、更好基础的路径。

## Abstract
Vision Language Models (VLMs) achieve strong reasoning with Chain-of-Thought (CoT) prompting but incur high sequential-generation cost, error accumulation, and limited self-correction. Diffusion Multimodal Large Language Models (dMLLMs) unmask tokens in an order-agnostic process, improving efficiency and enabling iterative refinement, yet their reasoning and how to enhance it remain underexplored. We propose a training-free method, Spatio-Temporal Token Veto (ST-Veto), which leverages the ability to observe all token positions at each diffusion step. Rather than relying only on current-step confidence, ST-Veto vetoes temporally unstable tokens via second-order Taylor prediction of confidence dynamics and filters weakly grounded tokens using image-attention mass, swapping them with safer candidates. Across multiple dMLLMs and multimodal reasoning benchmarks, ST-Veto consistently outperforms standard decoding policies and prior VLM reasoning methods, improving accuracy by up to 9% with no additional training or generation cost. Analyses show that ST-Veto steers generation toward higher-confidence, better-grounded paths.

# 2. 论文提出的方法论

- **核心思想**：ST-Veto（Spatio-Temporal Token Veto）在每一步去噪时，不仅依赖当前置信度，还结合时间稳定性和视觉基础两个信号，对高置信度但不可靠的令牌进行“否决”，并替换为更安全、稳定且有视觉支撑的候选令牌。整个过程在现有解码框架内可即插即用。
- **关键技术细节**：
    1. **泰勒残差（Temporal Instability Signal）**：利用二阶泰勒展开，从4步历史置信度（t-3至t步）预测当前步预期置信度。实际置信度显著低于预期（即负残差）的令牌被视为时间不稳定，触发否决。
        - 公式：\( \hat{c}^{(t)} = c^{(t-1)} + (c^{(t-1)}-c^{(t-2)}) + \frac{1}{2}[(c^{(t-1)}-c^{(t-2)}) - (c^{(t-2)}-c^{(t-3)})] \)，残差 \( e_i = c_i^{(t)} - \hat{c}_i^{(t)} \)。
    2. **图像注意力（Visual Grounding Signal）**：聚合模型中间层（1/3、1/2、2/3深度层）中令牌对图像令牌的注意力占比，作为视觉基础度量。注意力占比低的令牌被视为弱基础，触发否决。
    3. **否决-交换策略（Veto-and-Swap）**：
        - 候选集：当前步置信度最高的前k个令牌（即top-k）。
        - 否决条件：任何tokens满足 \( e_i < \mu_e - \sigma_e \)（不稳定）或 \( m_i < \mu_m - \sigma_m \)（弱基础），则从top-k中移除。
        - 安全条件：\( e_i \ge \mu_e + \sigma_e \) 且 \( m_i \ge \mu_m + \sigma_m \)。
        - 交换来源：不在top-k但属于安全条件且置信度在接近阈值范围内（\( |c_i^{(t)} - c_{(k)}| \le \sigma_c \)）的令牌，按置信度排序填充否决空缺。
        - 若安全候选数量不足，则仅执行部分交换。

# 7. 优点

- **训练-free且即插即用**：无需额外训练或模型修改，可直接集成到现有dMLLM推理流程。
- **零额外生成成本**：不使用二次推理调用，计算开销极小（仅增加轻量数学计算和注意力聚合）。
- **利用扩散独有优势**：能够观察所有令牌位置的演化，而非AR的顺序依赖。
- **稳健的否决策略**：采用基于中位数和MAD的鲁棒统计阈值，避免依赖绝对阈值，适应不同分布。
- **可视化分析一致性好**：置信度和注意力动态分析均支持方法有效性，不仅在指标上，也在内部机制上解释。
- **高度模块化**：可与不同remasking策略结合，未来可扩展至其他扩散语言模型。

# 8. 不足与局限

- **依赖内部信号**：置信度和注意力是正确性的不完美代理，非平滑置信度轨迹可能影响泰勒预测的准确性，图像注意力也可能与任务所需视觉证据不一致。
- **保守性限制**：泛化到多图像、视频或超长生成时，视觉令牌数量增加可能导致注意力稀释，需要额外设计归一化策略。
- **当前仅聚焦单图像多模态**：未验证在纯文本扩散模型或多轮对话场景。
- **否决-交换可能失败**：当安全候选不足时，ST-Veto只能执行部分交换，此时效果受限。
- **未讨论模型规模或训练数据**：实验结果基于特定两个dMLLM，暂未证明在更大/更小模型上的一致性，可推广性待验证。
- **潜在应用风险**：虽然方法提升准确性，但无法消除偏见、错误或恶意使用，部署时仍需人工监督。
